# Review a skill

**Layer:** proposed architecture. **Mode:** guide.

1. Start with [`reference/review-principles.md`](../reference/review-principles.md): justify the skill, check its shape and scope, prune bloat, and verify the phrasing before scoring.
2. Does it satisfy the fundamentals? Use [`fundamentals/architecture/evaluation.md`](../fundamentals/architecture/evaluation.md) and [`reference/audit-rubric.md`](../reference/audit-rubric.md).
3. Is it the right type? See [`../fundamentals/architecture/types/`](../fundamentals/architecture/types/).
4. Is it bloated? See [`fundamentals/core/common-mistakes/`](../fundamentals/core/common-mistakes/).
5. Does it adopt architecture patterns correctly? See the relevant docs in [`patterns/`](../patterns).
