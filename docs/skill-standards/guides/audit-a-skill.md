# Audit a skill

Use this guide to check a skill against the fundamentals and standards.

1. Read the skill's `SKILL.md` and identify its type and invocation mode.
2. Check it against [`reference/audit-rubric.md`](../reference/audit-rubric.md).
3. Review the workflow in [`reference/review-principles.md`](../reference/review-principles.md).
4. If it fails, report blockers, warnings, and suggestions with a clear remediation plan.
