# Dependencies

## Required skills

- `worker-contract` — provides the standard return format for adapters.
- `context-reports` — shared conventions for context-report adapters.
- `token-resolver` — token resolution order for adapters.

## No runtime tools

This building block is a reference contract. It does not invoke tools or scripts.

## Consumers

Conductors and adapters that implement this contract. Examples include `pr-report` and `pr-review`.
