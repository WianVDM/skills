# Session handoff

## Goal
One sentence: what the session is trying to achieve.

## Current state
- What is true now.
- What has been completed.

## Next action
- The single thing to do next.

## Tried approaches
- What was tried and what the outcome was.

## Blockers
- None, or the specific blocker and what is needed.

## User preferences
- Constraints or preferences the user stated.

## Links
- `path/to/file` — one-line description of what it is.

## Chain
- Link to the previous handoff or parent context, if any.
