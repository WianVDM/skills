# Token Resolver Dependencies

## Required skills

- `worker-contract` — provides the canonical worker return format.

## Required tools

- `read`, `write`, `edit` for inspecting config files.
- `bash` for running the optional helper script.

## Required binaries

- Python 3.x for the token-extraction script.

## Environment

- The token resolver only reads environment variables referenced by the caller. It does not require specific env vars itself.
