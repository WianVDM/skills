- `worker-contract` — canonical worker return format referenced by adapters and the conductor.

- `token-resolver` — resolves tokens without exposing them.
- `context-reports` — shared context-report conventions.

- `read` — to inspect this contract.

## Optional skills

- `pr-report` — consumes this contract. Not required for the contract itself.
