# Dependencies

## Required skills

- **list-available-skills** — discover skills already available in the project and user scope.
- **search-skills-registry** — find third-party skills in configured registries.
- **context-reports** — shared context-report conventions and schemas.

## Required tools and capabilities

- **Read files** in the skills directory and context directory.
- **Write files** in the context directory.
- **Run scripts** in the building-block skill directories.
- **Spawn subagents** if needed for focused classification.

## Required binaries

- `python3` (or equivalent) for running building-block scripts.

None.

None.

None directly; dependencies are owned by the building-block skills above.
