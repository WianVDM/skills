# Dependencies

## Required skills

None. `audit-skill` is a standalone building block.

## Required tools and capabilities

- **Read filesystem** — reads `SKILL.md`, `README.md`, references, subagents, and scripts from the skill under audit.
- **Python 3.x** — the script is written in Python and uses only the standard library.

## Required binaries

- `python3` (or equivalent Python 3 interpreter)

## Required MCP servers

None.

## Environment variables

None.

## Optional dependencies

None. The script uses only `pathlib`, `argparse`, `json`, and `re` from the Python standard library.
