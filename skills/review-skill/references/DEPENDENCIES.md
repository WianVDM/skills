# Dependencies

## Required skills

- **audit-skill** — evaluate a skill against the fundamentals rubric.
- **validate-skill-frontmatter** — validate `SKILL.md` frontmatter against the JSON schema.
- **context-reports** — shared context-report conventions and schemas.

## Required tools and capabilities

- **Read files** in the target skill directory and context directory.
- **Write files** in the context directory.
- **Edit files** in the target skill directory after approval in the `update` gate.
- **Run scripts** in the building-block skill directories.
- **List directories** to discover skill files.

## Required binaries

- `python3` (or equivalent) for running building-block scripts.

## Required MCP servers

None.

## Environment variables

None.

## Required Python packages

None directly; dependencies are owned by the building-block skills above.
